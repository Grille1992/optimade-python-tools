    Links

    ¶

    LinksResource pydantic ¶

    A Links endpoint resource object

    attributes: LinksResourceAttributes field pydantic required ¶

    a dictionary containing key-value pairs representing the entry's properties.

    id: str field pydantic required ¶

    An entry's ID as defined in section Definition of Terms_. - Type: string. - Requirements/Conventions:

    • Support: REQUIRED, MUST NOT be :val:null.
    • Query: MUST be a queryable property with support for all mandatory filter features.
    • Response: REQUIRED in the response.

    • See section Definition of Terms_.

    • Examples:

    • :val:"db/1234567"

    • :val:"cod/2000000"
    • :val:"cod/2000000@1234567"
    • :val:"nomad/L1234567890"
    • :val:"42"

    links: ResourceLinks field pydantic ¶

    a links object containing links related to the resource.

    meta: Meta field pydantic ¶

    a meta object containing non-standard meta-information about a resource that can not be represented as an attribute or relationship.

    relationships: EntryRelationships field pydantic ¶

    a dictionary containing references to other entries according to the description in section Relationships_ encoded as JSON API Relationships <https://jsonapi.org/format/1.0/#document-resource-object-relationships>__. The OPTIONAL human-readable description of the relationship MAY be provided in the :field:description field inside the :field:meta dictionary.

    type: str field pydantic ¶

    These objects are described in detail in the section Links Endpoint

    LinksResourceAttributes pydantic ¶

    Links endpoint resource object attributes

    base_url: Optional[Union[pydantic.networks.AnyUrl, optimade.models.jsonapi.Link]] field pydantic required ¶

    JSON API links object, pointing to the base URL for this implementation

    description: str field pydantic required ¶

    Human-readable description for the OPTIMADE API implementation a client may provide in a list to an end-user.

    homepage: Optional[Union[pydantic.networks.AnyUrl, optimade.models.jsonapi.Link]] field pydantic required ¶

    JSON API links object, pointing to a homepage URL for this implementation

    link_type: str field pydantic required ¶

    The link type of the represented resource in relation to this implementation. MUST be one of these values: 'child', 'root', 'external', 'providers'.

    name: str field pydantic required ¶

    Human-readable name for the OPTIMADE API implementation a client may provide in a list to an end-user.
